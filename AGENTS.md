# Agent Instructions for This Repository

This repository contains a reusable AI agent skill. Keep the skill focused on building and implementing modular, robust architecture for a concrete target.

When changing the repository:

- Keep `SKILL.md` concise enough for an agent to load and follow reliably.
- Preserve the activation boundary. The skill must not become a global coding style guide.
- Preserve the conversation-first workflow. The agent must ask targeted questions before scanning or editing.
- Preserve the two-stage question model: initial purpose questions, then evidence-based follow-up questions after scanning.
- Keep examples generic and reusable across software projects.
- Do not introduce domain-specific use cases unless they represent a common software architecture scenario.
- Do not reference private projects, customer code, or repository-specific implementation details.
- Update README examples when the skill behavior changes.
- Keep author attribution as Byron Jacobs / @savedpixel unless ownership changes.
